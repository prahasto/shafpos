2. Basic parsing of a JSON file

Here is a basic example of parsing a json file. You change the “posts.json” by the full url, ex: http://www.domain.tld/posts.php?format=json

  1. <?php
  2. // copy file content into a string var
  3. $json_file = file_get_contents('posts.json');
  4. // convert the string to a json object
  5. $jfo = json_decode($json_file);
  6. // read the title value
  7. $title = $jfo->result->title;
  8. // copy the posts array to a php var
  9. $posts = $jfo->result->posts;
  10. // listing posts
  11. foreach ($posts as $post) {
  12.     echo $post->title;
  13. }
  14. ?>
